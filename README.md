# 🧩 BuildTrack CLI – Project Management Tool

## 🧠 Project Overview

BuildTrack CLI is a Python-based command-line application that helps manage users, projects, and tasks.

It is designed using Object-Oriented Programming (OOP) principles and demonstrates:
- Class relationships (User → Project → Task)
- File-based data storage using JSON
- Command-Line Interface (CLI) interaction using Python
- Modular code structure for scalability and reusability

---

## 🚀 Features

### 👤 User Management
- Create new users with name and email
- Store user information persistently

### 📁 Project Management
- Assign projects to users
- Each user can have multiple projects
- View all projects linked to a user

### ✅ Task Management
- Add tasks to projects
- Assign status to tasks (e.g. pending, completed)
- Mark tasks as complete

### 💾 Data Persistence
- All data is stored using JSON files
- Data is loaded automatically when the app runs
- Changes are saved to file to ensure persistence
